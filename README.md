# AdventureWorks CRM Analytics Dashboard

## Goal

Analyze customer purchasing behavior and predict:

1. Buy / Not Buy (Classification)
2. Sales Amount (Regression)

using Random Forest algorithms from a CRM perspective.

## Data Preprocessing

* Missing Value Removal
* Duplicate Removal
* Outlier Removal

## Exploratory Data Analysis

* Revenue by Country
* Revenue by Product Category
* Revenue Trend by Year
* Customer Segmentation

## Machine Learning

### Classification

RandomForestClassifier

Predict whether a customer will buy products.

### Regression

RandomForestRegressor

Predict expected sales amount.
